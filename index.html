Recent Projects

  • Wanderer

    Wanderer is a website designed for travellers and travel enthusiasts who can share their travel stories with other fellow 'Wanderers'. They can interact with other travel posts and get travel information about areas their unvisited locations and plan their travel journey accordingly. The website also provides a feature for service providers in an area to advertise their services for all wanderers and grow their outreach.

  • Dungeon Ride

    Dungeon Ride is an interactive game developed using Java demonstrating the use of good design principles. The project uses MVC design pattern with model, view and controller working almost independently. The project uses Kruskal's algorithm for maze generation and player moves along the dungeon, collecting treasure, fighting monsters and eventually finding his/her path towards the exit.

    This is a personal project that visualizes various sorting algorithms and is developed using JavaScript, React, HTML and CSS.
